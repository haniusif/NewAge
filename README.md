# Laravel NewAge Starter 

Laravel Starter With Pusher , Firebase And Passport 

## Installation



#steps

```bash
$ git clone https://github.com/haniusif/newage.git
$ cd newage
$ composer install
$ npm install
$ npm run dev
$ npm run watch


$ cp .env.example .env
$ php artisan key:generate
$ php artisan serve
```
